About zStudio

What is zStudio?

zStudio is a lightweight visual authoring tool built to make 2D web game production faster, cleaner, and far more efficient. Designed by a game development lead with over 15 years of industry experience, it gives teams a powerful way to build UI layouts, scene structures, templates, and multi-resolution game interfaces — all visually, all in one place.

Originally created to solve real workflow bottlenecks inside a commercial game studio, zStudio is now used in production on multiple web-based titles, including games currently deployed to international operators. It is actively used by professional developers and continues to evolve through real-world feedback.

Why zStudio was built

Modern web games rely on responsive layouts, reusable UI systems, and efficient pipelines. But many studios still build and manage scenes manually — causing slow iteration, inconsistent layouts, and extra engineering overhead.

zStudio was created to bridge the gap between design and implementation. It gives developers, technical artists, and animators a unified space to compose scenes, control layout behavior, test responsiveness, and export clean, structured data directly into Pixi.js or similar engines.

What zStudio enables

  • Rapid UI/Layout creation — Build scenes visually instead of coding containers by hand.
  • Automatic multi-resolution support — Preview and optimize layout for portrait, landscape, and custom ratios.
  • Hierarchical scene building — Create, group, nest, and organize complex structures with ease.
  • Template-based workflows — Build reusable components that update all instances when changed.
  • Integrated test mode — Preview animations, hierarchies, and layout behavior instantly.
  • Efficient exporting — Generate JSON scene files, texture atlases, and font definitions ready for runtime.
  • Import pipeline — Bring zStudio scenes directly into your rendering environment.

Who it's for

zStudio is ideal for:

  • Web game developers building Pixi.js, Phaser, or custom 2D engines
  • Technical artists who need precise control without writing boilerplate code
  • UI/UX teams creating dynamic, responsive interfaces
  • Studios looking to streamline iteration and reduce integration time

A production-proven tool

zStudio is not a prototype — it is actively used in a commercial studio and has already powered multiple shipped and soon-to-launch titles. With continuous input from full-time game developers, the tool is shaped by real production needs, not theoretical use cases.

The vision

zStudio aims to become the standard visual layout and scene-building tool for web game development — a fast, accessible, and scalable environment that integrates seamlessly into modern pipelines.
